1. В базе данных Redis подберите коллекцию для подсчета посещений с определенных IP-адресов.

Создадим список ip_list, содержащий ip адреса
RPUSH ip_list 192.168.1.2 192.168.1.3 192.168.1.5 192.168.1.4 192.168.1.7 192.168.1.8

Посчитаем длину списка
LLNE ip_list




2. При помощи базы данных Redis решите задачу поиска имени пользователя по электронному адресу и наоборот, поиск электронного адреса пользователя по его имени.

Создаём пару ключ-значение имя-почта
SET Ivan ivanov@mail.ru

Создаём пару ключ-значение почта-имя
SET ivanov@mail.ru Ivan

Ищем имя по почте
GET ivanov@mail.ru

Ищем почту по имени
GET Ivan
